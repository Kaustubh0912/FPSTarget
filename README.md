# FPSTarget

A sophisticated First-Person Shooter (FPS) targeting practice project built with Unity, featuring advanced player controls, realistic weapon mechanics, and dynamic targeting systems.

## Features

### Advanced Player Movement
- Smooth WASD movement with acceleration and momentum
- Sprint capability (Left Shift)
- Crouch mechanics (C key)
- Jump functionality (Space)
- Air control for mid-jump movement
- Custom movement curve for responsive feel

### Weapon System
- Configurable firing modes (automatic/semi-automatic)
- Realistic ammunition system with reload mechanics (R key)
- Advanced recoil patterns with customizable behavior
- Weapon sway for immersive feel
- Muzzle effects and impact visualization
- Adjustable accuracy system based on movement state
- Auto-reload option when empty

### Target Practice Elements
- Bullseye targets with multiple scoring zones
- Dynamic target system with customizable points
- Destructible objects
- Real-time score tracking
- Hit feedback system
- Visual hit markers and effects

### Performance Features
- Optimized raycasting system
- Efficient particle systems
- Smooth camera controls with sensitivity settings
- Modular code architecture for easy expansion

## Controls

- **Movement**: WASD
- **Sprint**: Left Shift
- **Crouch**: C
- **Jump**: Space
- **Shoot**: Left Mouse Button
- **Reload**: R
- **Mouse**: Look/Aim

3. Open the main scene from `Assets/Scenes`
4. Press Play to test the game
